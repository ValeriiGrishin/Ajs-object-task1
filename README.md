## Описание задания

Реализация функции `orderByProps`, которая для переданного объекта возвращает массив его свойств со значениями, отсортированный по свойствам.

## Требования:
- Использование `for...in` для перебора свойств объекта
- 100% покрытие тестами (unit tests)
- Прохождение ESLint без ошибок
- Сборка на базе Webpack

## Результат:

- Функция orderByProps с обработкой ошибок;
- 13 unit-тестов с полным покрытием (100% Statements, Branch, Functions, Lines)
- Настроен CI/CD с автоматической генерацией бейджей покрытия
